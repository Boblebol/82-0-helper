# Changelog

All notable changes to this project are documented here.

This project follows semantic versioning for public releases.

## [0.1.1] - 2026-06-17

### Changed

- Bumped the package and extension manifest version for the public release.

### Fixed

- Kept help tooltips inside sidebar cards instead of overflowing outside the panel.

## [0.1.0] - 2026-06-17

### Added

- Classic mode recommendation sidebar for 82-0.com.
- Current roll detection for team, decade, round, roster, and visible players.
- Manual page scan action and manual roster editing.
- Estimated game-score impact per pick.
- Reroll advice with better-pick probabilities.
- French UI with contextual help content.
- Light theme matching the 82-0.com page.
