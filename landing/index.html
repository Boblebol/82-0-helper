82-0 Helper

Une sidebar française pour choisir le meilleur pick, estimer le score de ta team et savoir quand garder tes rerolls.

Télécharger l'extension Guide d'installation

Aucune automatisation: l'extension conseille, tu gardes la main sur chaque choix.

Lit le roll team, décennie, round et roster visible.
Compare les picks impact estimé sur le score final.
Aide au reroll proba de trouver mieux avec tes skips restants.

But du jeu

82-0 te met au défi de construire un roster NBA historique invaincu.

À chaque round, le site te donne une équipe et une décennie. Ton objectif est de choisir le joueur qui renforce le mieux ton cinq majeur pour viser une saison parfaite.

82-0 Helper ne joue pas à ta place: il lit le roll visible, compare les options et t'aide à décider si le pick ou le reroll vaut le coup.

Installation rapide

Guide d'installation

L'extension n'est pas encore sur le Chrome Web Store. Tu l'installes en mode développeur depuis le zip de la release GitHub.

Télécharger l'extension
  1. Télécharge le zip Récupère la dernière release puis dézippe le dossier sur ton ordinateur.
  2. Ouvre chrome://extensions Ça marche aussi dans Edge et Brave avec leur page extensions équivalente.
  3. Active le Mode développeur Le bouton est en haut à droite sur Chrome.
  4. Charge le dossier dézippé Clique sur "Charger l'extension non empaquetée", puis sélectionne le dossier de l'extension.
  5. Lance une partie Classic Va sur 82-0.com, spin, puis laisse la sidebar te conseiller.
