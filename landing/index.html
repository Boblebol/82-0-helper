Installation rapide

Guide d'installation

L'extension n'est pas encore sur le Chrome Web Store. Tu l'installes en mode développeur depuis le zip de la release GitHub.

Télécharger l'extension
  1. Télécharge le zip Récupère la dernière release puis dézippe le dossier sur ton ordinateur.
  2. Ouvre chrome://extensions Ça marche aussi dans Edge et Brave avec leur page extensions équivalente.
  3. Active le Mode développeur Le bouton est en haut à droite sur Chrome.
  4. Charge le dossier dézippé Clique sur "Charger l'extension non empaquetée", puis sélectionne le dossier de l'extension.
  5. Lance une partie Classic Va sur 82-0.com, spin, puis laisse la sidebar te conseiller.

Open source

Simple, local, transparent.

Pas de compte, pas de serveur, pas de télémétrie. Le code est public et l'extension ne fait que lire la page pour afficher ses conseils.
